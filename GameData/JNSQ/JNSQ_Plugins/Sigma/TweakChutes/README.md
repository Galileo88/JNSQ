### How To Install

To install the mod, follow these instructions:

```
>> Install Manually
  - Remove any other version of the mod.
  - Unzip the archive directly into the folder 'KSP\GameData'

>> Uninstall manually
  - Delete the folder 'KSP\GameData\Sigma\TweakChutes'
```
